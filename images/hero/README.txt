HERO IMAGE
==========

The hero section currently uses your existing food photo as a
darkened, atmospheric background:

    images/Whos doing it like this/IMG_1518.PNG

To swap it for a different shot, drop a new file in here as:

    images/hero/hero-plate.jpg

Then in css/styles.css search for "HERO BACKGROUND" and update the path.

Best results: a wide landscape food photo (1600px+) where the food
is roughly centered. The CSS adds a dark warm overlay so the headline
stays readable.
