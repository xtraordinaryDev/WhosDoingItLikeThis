LOGO FOLDER
===========

The site is currently using your existing logo file at:

    images/Whos doing it like this/logo.jpg

If you'd like to organize files cleanly, copy/move it here as:

    images/logo/logo.png   (or .jpg)

Then update the <img src="..."> references inside index.html
(search for "logo.jpg" -- there are 3 spots: nav, hero, footer).

A square or roughly square version works best for the small nav avatar.
